# The Crochet Sab - Handmade Custom Quality Designs

## 🧶 Welcome to The Crochet Sab

A beautiful e-commerce website showcasing handmade crochet products including keychains, hand gloves, bags, home decor, and custom designs.

## 🌐 Live Website

**Visit the website here:** [The Crochet Sab](https://imranraufbm.github.io/thecrochetsab/)

## 📂 Project Structure

```
thecrochetsab/
├── index.html              # Home page
├── categories.html         # Browse all categories
├── category.html          # Category product view
├── bags.html              # Bags gallery showcase
├── products.html          # Complete product catalog
├── contact.html           # Contact page
├── css/
│   └── style.css         # Main stylesheet
├── images/
│   ├── bags/             # Bag product images
│   ├── banners/          # Hero and section banners
│   ├── categories/       # Category thumbnails
│   ├── custom-designs/   # Custom design examples
│   ├── home-decor/       # Home decoration items
│   ├── keychains/        # Keychain designs
│   ├── logo/             # Brand logos
│   ├── products/         # Product showcase images
│   ├── wearables/        # Hand gloves and wearables
│   └── icons/            # UI icons
└── js/                    # JavaScript files

```

## 📦 How to Use (For Developers)

**Clone the repository:**
```bash
git clone https://github.com/imranraufbm/thecrochetsab.git
cd thecrochetsab
```

**Open in browser:**
- Simply open `index.html` in your web browser
- No build process or dependencies required
- All files are self-contained

**File Organization:**
- Relative paths used throughout for portability
- Works locally and on any hosting platform
- Easy to customize and extend

## 🔧 Customization

To customize the website:
1. Edit HTML files to change content
2. Modify `css/style.css` for styling
3. Replace images in respective folders
4. Update product data in JavaScript sections
5. Push changes to GitHub
